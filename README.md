# L-Corparation Website

Website giới thiệu sản phẩm xe điện của L-Corparation, được xây dựng với HTML, CSS, và JavaScript thuần.

## Tính năng chính

-   **Giao diện hiện đại:** Responsive design, hỗ trợ tốt trên cả Mobile và Desktop.
-   **Đa ngôn ngữ:** Hỗ trợ chuyển đổi qua lại giữa Tiếng Việt (VI) và Tiếng Anh (EN).
-   **Dữ liệu động:** Thông tin xe (Ô tô, Xe máy điện) được tách biệt trong các file JS, dễ dàng cập nhật và bảo trì.
-   **Trải nghiệm người dùng:**
    -   Mega Menu tích hợp Carousel.
    -   Modal thông số kỹ thuật chi tiết dạng Brochure.
    -   Hiệu ứng chuyển động mượt mà.

## Hướng dẫn Cài đặt & Git

### Kết nối với Repository

Nếu bạn đang ở thư mục dự án trên máy cục bộ, hãy chạy các lệnh sau để kết nối với GitHub:

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/nguyenmt0126/project-page
git push -u origin main
```

### Chạy dự án

1.  Tải mã nguồn về hoặc clone repository.
2.  Mở file `Pages/index.html` trong trình duyệt web bất kỳ.
3.  (Khuyến nghị) Sử dụng **Live Server** trên VS Code để có trải nghiệm tốt nhất.

## Cấu trúc Thư mục

-   `Pages/`: Chứa mã nguồn HTML chính.
-   `Css/`: Chứa các file định kiểu (Style).
-   `JavaScript/`: Chứa logic xử lý, dữ liệu xe và ngôn ngữ.
-   `Data/`: Chứa tài nguyên hình ảnh.
-   `DOCUMENTATION.md`: Tài liệu chi tiết về logic và kiến trúc hệ thống.
